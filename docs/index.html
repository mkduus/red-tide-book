Red Tide Data and Modeling Book

This Jupyter Book compiles the data processing, visualization, and machine learning analyses developed for the Southwest Florida red tide study.

The notebooks included here document:

  • Environmental and hydrologic data integration for the Peace and Caloosahatchee River systems

  • Watershed definitions and spatial context

  • Machine learning–based prediction of Karenia brevis bloom occurrence, including the incorporation of Tampa Bay observations

This book is intended to provide a transparent, citable record of the analytical workflow supporting the associated journal manuscript.
